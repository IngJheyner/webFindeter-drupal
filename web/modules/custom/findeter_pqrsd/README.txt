Findeter PQRSD Manager

INTRODUCTION
------------

This module built to implement custom ajax form for Findeter and custom administrators form, that create a new content type to manage all data. It was implemented for Drupal 8
and 9.

INSTALLATION
------------
 
 * Install as you would normally install a contributed Drupal module. 
   Visit:
    https://www.drupal.org/docs/extending-drupal
   for further information.
